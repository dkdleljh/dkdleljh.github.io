# Projects

현재 다루는 프로젝트를 한눈에 볼 수 있도록 정리한 페이지입니다. 아래에는 직접 소개하는 핵심 프로젝트를 먼저 두고, 맨 아래에는 GitHub 저장소 전체 목록을 자동으로 동기화해 보여 줍니다.

## 핵심 프로젝트

### Autonomous Open Problem Lab

수학 난제 후보를 수집하고, 정규화하고, 검증하고, 형식화와 논문화까지 이어 주는 무인 연구 파이프라인입니다.

- 링크: <https://github.com/dkdleljh/autonomous_open_problem_lab>
- 초점: 자동화 연구 워크플로우, 감사 로그, 게이트 기반 품질 관리

### My-Second-Brain (Private)

Obsidian Vault를 기반으로 기록, 연결, 실행 루프를 실제 운영 시스템처럼 관리하는 개인 지식 프로젝트입니다.

- 목표: 검색 가능한 기록 + 자동화된 운영
- 운영 원칙: 성공은 침묵, 실패만 알림 / 민감정보는 외부 공개 제외

### 이 블로그

GitHub Pages와 Jekyll 기반으로 운영하는 개인 브랜딩 사이트입니다. 단순 소개 페이지가 아니라 프로젝트 인덱스와 운영 흔적을 함께 쌓는 공간으로 사용하고 있습니다.

- 링크: <https://github.com/dkdleljh/dkdleljh.github.io>
- 역할: 포트폴리오, 링크 허브, 프로젝트 인덱스
